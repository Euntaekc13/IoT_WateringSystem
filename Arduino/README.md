# Arduino Area

## Arduino Board
- Arduino Uno

## Arduino device
- capacitive soil moisture sensor v1.2 : 습도 센서
- step motor : 모터
- ESP-01 : Wifi 장치

## Arduino Libraries used
```
ArduinoJson : 6.19.4
Stepper : 1.1.2
WifiEsp : 2.2.1
```

## Arduino setup
1. 아두이노 baud 9600으로 맞추기 
2. ESP-01 IP 알아내기
3. ESP-01 IP와 Wifi의 SSID와 PASSWORD 설정하기
4. 아두이노 UNO에 Device 연결하기
