# WateringSystem FrontEnd - Vue.js

## This repo is for Vue2
You are looking at the repository for Vue2.

## 라이브러리 버전
```
    "axios": "^1.1.3",
    "chart.js": "^4.0.1",
    "core-js": "^3.8.3",
    "vue": "^2.6.14",
    "vue-chartjs": "^4.1.2",
    "vue-router": "^3.5.1",
    "vuetify": "^2.6.0",
    "vuex": "^3.6.2"
```

## Vue CLI 설치
g: 해당 패키지가 현재 디렉토리뿐만 아니라 앞으로 생성되는 모든 프로젝트에서 사용할 수 있는 global 패키지 등록

```console
> npm install -g @vue/cli
```

## Vue 프로젝트 생성
```console
> vue create vue-project 
```

## Vue 프로젝트 실행
```console
> npm run serve 
```

## Vuetify 
```console
> npm install vuetify
```

## chart.js
(https://www.chartjs.org/docs/latest/getting-started/installation.html)
```console
> npm install chart.js
```

## Font Awesome
SVG Core 추가 (Add SVG Core)
```console
> npm i --save @fortawesome/fontawesome-svg-core
```
아이콘 패키지 추가 (Add Icon Packages)
```console
> npm i --save @fortawesome/free-solid-svg-icons
> npm i --save @fortawesome/free-regular-svg-icons
> npm i --save @fortawesome/free-brands-svg-icons
```
Vue Component 추가 (Add the Vue Component)
```console
# for Vue 2.x
> npm i --save @fortawesome/vue-fontawesome@latest-2
```
